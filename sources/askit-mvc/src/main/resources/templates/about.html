About

What is Askit?

Askit is a question and answer Website designed for educational purposes in which you can help and be helped or guided by others to find a solution.

Why Askit?

Askit offers an unique experience in which you can interact with the super-intuitive UI and visualize the content you need.

How was Askit created?

Askit was created using a variety of technologies and programming languages, but the most important of all are:

Spring
Spring is a framework used for the backend where all the logic is done.
